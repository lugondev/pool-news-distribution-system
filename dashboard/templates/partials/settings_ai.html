Topic Synthesis — AI analyzes multiple articles and generates diverse summaries

✨ New Feature
Instead of translating articles one-by-one, AI can group articles by category and generate 1-8 synthetic summaries with different angles (timeline, analysis, comparison, impact). AI autonomously decides output count based on content diversity.
Example
// Input: 5 politics articles about "US-China trade tensions"
→ AI generates 5 outputs:
1. Timeline summary
2. US perspective
3. China perspective
4. Economic impact
5. Tech sector implications

How it works

// Input: article from Redis (ai_status = "pending")
"title": "Original article title"
"content": "Article body text (first 1500 chars)"

// POST → {{ ai.base_url or 'OPENAI_BASE_URL env' }}/chat/completions
// Model: {{ ai.model or 'gpt-4o-mini' }} | temp: {{ ai.temperature if ai.temperature is not none else 0.3 }} | tone: {{ ai.tone or 'general' }} | format: json_object
// Prompt tone: {% if (ai.tone or 'general') == 'formal' %}"Serious, authoritative news editor"{% elif (ai.tone or 'general') == 'casual' %}"Friendly, upbeat news writer"{% else %}"Professional news editor — clear, neutral"{% endif %}

// Output: stored back to Redis article hash
"ai_summary_vi": "Tóm tắt tiếng Việt (2-3 câu)"
"ai_summary_en": "English summary (2-3 sentences)"
"ai_status": "done"
