Setup Guide

1. Message @BotFather on Telegram → /newbot → copy the token
2. Create a channel/group, add the bot as admin
3. Get chat ID: forward a message from the channel to @userinfobot, or use https://api.telegram.org/bot{TOKEN}/getUpdates
4. Add the channel here and click Test to verify
